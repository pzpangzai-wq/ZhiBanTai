# 发布说明

## v0.1.2

- 修复部分局域网中网关或网络设备响应多个 IP 端口，导致首次扫描误生成 20 张电脑卡片的问题
- 自动发现新增卡片时改为以真实局域网 ARP/MAC 记录为准，避免把纯 IP 端口响应误记为电脑
- 保留已有配置设备的在线探测能力，已手动配置或已记住的 IP 仍可正常检测

## v0.1.1

- 修复首次启动局域网扫描可能长时间停留在“首次扫描”界面的问题
- 为整次局域网扫描增加总超时，网络环境异常时也会回到主界面
- 为主机名反查增加超时，反查缓慢时自动使用 IP 显示设备
- 限制扫描并发数量，降低首次启动时对局域网和系统线程的压力

## v0.1.0

- 初版值班台程序，面向 Win7/Windows 局域网查询检测场景
- 支持局域网扫描、设备记忆、离线卡片保留
- 支持电脑共享访问状态提示
- 支持网络打印机识别和打印机卡片
- 支持卡片重命名、删除和二次确认
- 支持托盘运行、开机自启动和静默启动
