# 值班台

值班台是一款面向 Win7/Windows 局域网环境的小型查询检测工具。它用卡片方式展示同一局域网内的电脑和常见网络打印机，适合值班室、办公室、小型机房等仍有 Windows 7 设备的场景。

程序会自动扫描局域网设备，记住曾经发现过的电脑或打印机；设备离线后仍会保留卡片，方便值班人员快速判断当前在线状态。点击在线电脑卡片可以打开 Windows 共享，点击打印机卡片可以尝试打开设备页面。

## 功能特点

- 自动扫描局域网设备，最多显示 20 台远程设备
- 兼容 Win7 使用环境，基于 .NET Framework WinForms 实现
- 记住扫描到过的电脑或打印机，离线后保留卡片
- 支持手动扫描，随时刷新在线状态
- 点击在线电脑卡片打开 Windows 共享
- 根据共享访问结果显示绿色“直达”或黄色“已提醒”
- 识别常见网络打印机，并使用打印机卡片样式显示
- 支持卡片重命名和删除，删除前会二次确认
- 支持托盘运行、开机自启动、静默启动
- 单文件 WinForms 程序，无需安装即可运行

## 快速使用

1. 下载发布包或自行构建程序。
2. 双击 `ZhiBanTai.exe` 或发布包中的 `值班台.exe`。
3. 首次启动会自动扫描局域网。
4. 需要刷新时点击右上角“扫描”。
5. 点击在线电脑卡片可以打开对方共享。

## 构建

项目是单文件 C# WinForms 程序，可使用 Windows 自带 .NET Framework C# 编译器构建：

```powershell
PowerShell -ExecutionPolicy Bypass -File .\build.ps1
```

构建成功后会生成：

```text
ZhiBanTai.exe
```

## 配置文件

程序会在同目录生成以下运行数据文件：

- `电脑配置.txt`：设备卡片配置和记忆数据
- `电脑清单.txt`：当前设备清单导出
- `程序设置.txt`：托盘、开机自启动等设置
- `错误日志.txt`：异常日志

公开仓库只保留 `电脑配置.example.txt` 和 `程序设置.example.txt`，避免上传真实电脑名、IP、本机路径或局域网信息。
